O Software Latex

10/06/2022

LaTex (abreviação Lamport Tex), nada mais é que uma composição de muita qualidade, tendo recursos planejados para produzir uma d>E também é possível o produzir em vários tipos de documentos, como PDF, HTML, PS, ABNT duas colunas etc.

Pontos Positivos

  • Como o mesmo possui um algoritmo avançado, permite a criação de documentos com aparência profissional.
  • Não possui bugs
  • Versões do APP não possuem incompatibilidade.
  • Os documentos Tex são leves e de fácil exportação.
  • Não é necessária muita pesquisa para conseguir criar, pois não existem muitos códigos.
  • Ideal para documentos com muitas páginas, por conta da leveza.
  • Templates mais consistentes e seguros;
  • Possibilidade de incorporação vídeos, áudio e animações.
  • Mas não é um processador de texto igual OpenOffice ou o Word do Office.
